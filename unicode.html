How to use Unicode Explorer

Unicode Explorer
  1. First open the unicode explorer from the button above, after going there you will see plenty of unicode values.
  2. You can scroll to find plenty of unicode values, ranging from E000 to EFFF. Find the unicode value you want to copy and click it.
  3. After clicking the unicode value you will see a few buttons. Click copy, it should say "Copied to clipboard".
  4. Open Minecraft and then join CraftersMC, In a book and quill (purchasable from the Librarian NPC) paste the unicode value(s) that you copied.

Direct Copy Unicode Values

Irrl's Glyph

After clicking on the button above, you will be redirected to a site where all glyphs(All Icons Simply) will be available to copy instantly one by one without needing to use Unicode Explorer. Please wait for a while so all the glyphs can load properly. After the glyphs are loaded, click on those you want and they will be copied to your clipboard instantly and then you can paste them in your book n quill in CraftersMC.

Notes

Not all unicode values will work, E400-407 are the store ranks whereas E420-425 are staff ranks. If you see nothing appear, you can try a different unicode value. That may work instead.

Pasting "E400" etc will NOT WORK. You MUST copy a UNICODE value using the websites given above.
